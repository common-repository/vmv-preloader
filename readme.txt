== Description ==

VMV Preloader is a modern solution with SVG/JavaScript/CSS animation. Preloader based on Web Components.  It offers visual feedback in the event of content being loaded.
This solution increase the chance of a new user wait load your Wordpress website.

VMV Preloader is:

* Full Responsive and retina ready
* No Image
* Faster Loading
* No coding necessary.

More information [vmv.one](https://vmv.one)

Right now plugin has 11 themes and one example of personal signature.

== Installation ==
1. Download and activate the plugin
2. Go to `settings` => `VMV Preloader`
3. Choose the Options of your choice
4. Click `Save` button.

Or

1. Put the plug-in folder `VMV Preloader` into [wordpress_dir]/wp-content/plugins/
2. Go into the WordPress admin interface and activate the plugin
3. Go to `settings` => `VMV Preloader`
4. Choose the Options of your choice
5. Click `Save` button.
